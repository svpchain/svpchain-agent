SVPChain Agent — Windows 运行前先阅读
========================================

本 zip 压缩包解压后包含文件夹：

  SVPChain Agent\
    svpchain-gui.exe     图形化配置工具（导入密钥、生成 MCP 配置）
    svpchain-mcp.exe     本地 MCP 签名服务（供 Cursor 等客户端调用）
    运行前先阅读.txt
    READ-BEFORE-RUN.txt  英文版说明（English guide）


一、安装步骤
------------

1. 将 zip 解压到任意目录（例如「下载」或 C:\Program Files）。
2. 进入解压后的「SVPChain Agent」文件夹。
3. 按下面「首次运行」方式启动 svpchain-gui.exe。
4. 在应用内完成：导入密钥 → 生成 MCP 配置 → 复制到 Cursor 等客户端设置。

请保持 svpchain-gui.exe 与 svpchain-mcp.exe 在同一文件夹内，GUI 会自动找到签名程序。


二、首次运行（重要）
--------------------

Windows SmartScreen 可能提示「未知发布者」。未购买商业代码签名证书的
开源软件出现此提示属于正常现象。

  1. 若被拦截，可点「更多信息」→「仍要运行」
  2. 或右键 svpchain-gui.exe → 属性 → 勾选「解除锁定」→ 确定，再运行

密钥保存在 Windows 凭据管理器中，不会写入明文配置文件。


三、MCP 客户端配置
-------------------

GUI 生成的 MCP JSON 会包含 svpchain-mcp.exe 的绝对路径。
若移动了安装文件夹，请在 GUI 中重新生成配置以更新路径。


四、配置完成后
--------------

GUI 主要用于首次配置。日常使用 Cursor 等 MCP 客户端时会自动通过 stdio
启动 svpchain-mcp.exe，无需一直开着 GUI。

应用内升级会替换你解压安装所在的文件夹（与 svpchain-gui.exe 同目录），
放在 D 盘或其他盘符均可。若升级后版本未变化，请查看日志：

  %LOCALAPPDATA%\com.svpchain.agent-gui\update\apply-update.log


五、系统要求
------------

  • Windows 10 或更高版本（64 位）
  • WebView2 运行时（多数系统已预装；缺失时 Wails 会提示安装）
  • 建议在 Release 页面对照 SHA256SUMS 校验下载文件
  • 请仅从官方 GitHub Release 下载，勿使用第三方重打包版本

若仍无法运行，请在项目 Issue 中附上 Windows 版本与错误截图。
